CNN

A class of Deep Neural Network most widely used approach to analyze visual imagery.

Mask Up - face mask detection software makes use of CNN i.e. Convolutional Neural Network. CNN has multiple convolution layers that are used to detect the facial features through the lines and edges detection. Convolution is a mathematical function / translation / combination of input data with the use of filter or kernel to provide a feature map.

CNN consists of input layer for taking inputs, multiple hidden layers for convolution processing and output layer for generating output.

Demystification

Machine Learning is one of the hot topics of the world. Image classifier is a part of machine learning. The most famous machine learning algorihtm for image classification is Convolutional Neyral Network i.e. CNN.

So, basically what CNN is - as we know its a machine learning algorithm for machines to understand the features of the image with foresight and remember the features to guess whether the name of the new image fed to the machine.

Training data sets are required to train the CNN model with the images. The more the training data sets, the more better will be the recognition capability of the software.

Now after getting the data set, the preprocessing of data is required and providing labels to each image to be able to detect the data from the test data.

The construction of CNN is a multi-layered feed-forward neural network, made by assembling many unseen layers on top of each other in a particular order. It is the sequential design that gives permission to CNN to learn hierarchical attributes. In CNN, some of them followed by grouping layers and hidden layers are typically convolutional layers followed by activation layers.

Residual network in CNN consists of a 3*3 convolution layer and a batch normalisation layer. IT also have skip connections called the identity connections, where the previous layers add up to the next layer, forming a residual block. Such residual blocks are repeated to form a residual network.

They have the ability to solve vanishing and exploding gradients when adding more layers to an already deep neural network. Residual Networks makes use of reLU activaion functions.

Activation functions decides whether a neuron should be activated or not, by calculating the weighted input and adding bias to it. Its purpose is to bring non-linearity in the output of neuron.

Benefits
  • Weight sharing
  • CNNs outperform NNs on conventional image recognition tasks and many other tasks
  • For a completely new task / problem CNNs are very good feature extractors.
Limitations
  • Fails to understand the contents of an image.
  • This limitation is more than evident when it comes to practical applications.
  • Several studies have shown that CNNs trained on ImageNet and other popular datasets fail to detect objects when they see them under different lighting conditions and from new angles.
