About us

The primary objective of our project is to develop a fully functional software that can identify if a person visible in an image or a video is wearing a mask or not.

Various independent evaluation tests have been done to develop this project and to carry out this project to completion.

As it is cumbersome for a human to accomplish this task and his/her eyes are bound to get tired, this software provides an alternate solution.
